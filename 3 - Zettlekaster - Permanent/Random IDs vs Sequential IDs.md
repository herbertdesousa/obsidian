Sequential IDs:
- **Fast insertions**: once b-tree structures don't need to **page split** reordering data in the leaf's.
- **Human readable**

Random UUID:
- **Distributed Systems**: Generate ID in multiple clusters, in same time.
- **Client ID generation**: Clients can create its id, being able to:
	- Deduplication and Idempotency
	- Offline Data Generation
